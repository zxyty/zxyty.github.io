linux系统nvm指定版本安装

二月 23, 2018

直接获取当前已存在的nvm版本

选择你想要安装的nvm版本

可以从nvm_github_release中找到最新的版本号, 如下:

这里查找到最新版本为0.34.0，然后执行命令

curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.34.0/install.sh | bash

最后导出环境变量

source ~/.bashrc

查看是否安装成功

nvm --version

#echo 0.34.0

ok.

续

使用指定版本的node

nvm ls-remote   # 获取远程node版本

nvm install v10.15.1    # 安装指定版本node

nvm alias default v10.15.1  # 设置v10.15.1为node默认版本

nvm use v10.15.1    # 使用v10.15.1 版本的node
